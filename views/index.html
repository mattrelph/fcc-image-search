Free Code Camp - Image Search Abstraction Layer

API that fufills these user stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

To search for images:

https://colossal-prose.glitch.me/api/imagesearch/cute%20bunnies?offset=2
https://colossal-prose.glitch.me/api/imagesearch/giant%20robots?offset=10

To view recent searches:

https://colossal-prose.glitch.me/api/latest/imagesearch/

*Please note: Only 100 searches allowed per day.
